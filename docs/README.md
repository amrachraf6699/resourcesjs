# ResourcesJS Documentation Site

This folder contains the Astro Starlight documentation site for ResourcesJS.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

The site is configured for:

```txt
https://resourcesjs.amrachraf.cloud
```

## Deployment

Use either GitHub Pages or Cloudflare Pages.

Cloudflare Pages settings:

- Root directory: `docs`
- Build command: `npm run build`
- Build output directory: `dist`
- Node version: `22.12.0` or newer

GitHub Pages is also configured through `.github/workflows/docs.yml`.
